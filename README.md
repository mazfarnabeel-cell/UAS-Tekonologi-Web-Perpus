fitur Aplikasi 
1. CRUD untuk menambahkan, mengedit, dan mengapus buku 
2. API terintegrasi 
3. pop up ketika berhasil menambahkan buku / eror 

Struktur folder 
A. backend 
- server.js 
B. publik 
- Index.html 
- style.css
- script.js

cara menjalankannya
1. kirim perintah node backend/server.js pada terminal 
2. buka https://localhost:3000 pada web broser 
3. terbuka tampilan Aplikasi nya, dan bisa
